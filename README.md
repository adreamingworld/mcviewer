# MCViewer

Simple map viewer for minecraft.
![Screenshot](resources/screenshot1.png?raw=true)

## Compile

```
./autogen.sh
./configure
make
```

## Running
	./mcviewer resources/r.0.0.mca

	WASD		- Movement
	Q		- Move up along the Y-axis
	E		- Move down along the Y-axis
	Left-mouse	- Grab and rotate the view
	Space		- Place a block
